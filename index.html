🔥 Firebase Configuration

📋 Steps to get your Firebase config:

  1. Go to console.firebase.google.com
  2. Create a new project (or select existing)
  3. Click ⚙️ → Project Settings → General tab
  4. Under "Your apps", click Web (</>)
  5. Register app, then copy the firebaseConfig object
  6. In Firebase console, enable Authentication → Email/Password
  7. In Firestore Database, create a database (start in test mode)

Your config is saved locally in your browser. Without Firebase, the app falls back to local storage (single-device only).
